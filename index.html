Introduction

I chose to investigate card counting in blackjack for my computer science programming project mostly because it killed two birds with one stone. I’ve been interested in learning about how to count cards for a while now, but since it takes a lot of commitment to really get a handle on it, I’d been sceptical about how much time it would take away from me doing schoolwork. Especially since these few years I’m in have a lot of important exams around them. So getting to do this for my project means that I get to learn about it, but without feeling like that time would be better used for school, since I can study it for my Computer Science A Level.
